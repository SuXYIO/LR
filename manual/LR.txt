NAME
	LRDemo (Linear Regression Demo)
SYNOPSIS
	./LR.out [args]
DESCRIPTION
	Linear regression demo made with C standard libs. 
COMMANDS
	tr/train [args]
		Auto training. 
	! [cmd]
		Execute shell commands. 
	h/help/man [entry_name]
		Help (manual) page, prints manual for entry, prints menu for LR if entry_name empty. 
	v/ver/version
		Print version info. 
	q/quit/exit
		Quits program. 
AUTHOR
