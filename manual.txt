NAME
	LR (Linear Regression)
	Version 1.0.0
SYNOPSIS
	./LR.out [args]
DESCRIPTION
	Linear regression demo made with C standard libs. 
OPTIONS
	-q	Quiet mode, only prints end info. 
	-t	Thread mode, enable muti-thread. Might increase calculation speed if batch_size is large. 
	-s [seed]	Define seed, use the seed to generate random, can duplicate run. If not enabled, the program will seed using time. 
	-v	Print version info. 
	-h	Help (manual) page, prints this manual.
